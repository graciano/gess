Gess

Grid Extremely Simple for Sass. That's it.

Install

  • With bower: $ bower install gess
  • Or download the code and copy/paste the content of the src/ folder in any folder of your project.

Usage

Include the gess mixin with these arguments:

@include row('selector-of-row', <width, widhts, or yet a map explained bellow>, <size of device, for responsiveness>); //You'll have to import it: @import "path/to/gess";

/** * And use it like this: */

@include gess(50%, 'big'); @include gess(100%, 'small'); //Or use a list of widths: @include row((25%, 25%, 100%, 200px)); //Or a map with the keys as string selectors and values as widths. @include row(( '.info-header': 100%, '.call-us': 50%, '.address': 50% ), 'biggest');

Customization

You can customize the breakpoints (and/or names of devices) after you import gess:

$breakpoints: (320px, 600px, 1200px, 1900px); //and then: $devices: ( 'phone' : brk($breakpoints, 1), 'tablet' : brk($breakpoints, 2), );
